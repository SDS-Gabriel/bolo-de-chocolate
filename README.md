# bolo-de-chocolate
aula de culinária

📖 Conceitos Praticados na Aula
Em HTML5:
Cabeçalhos Semânticos: Organização hierárquica usando <h1>, <h2> e <h3>.

Listas Estruturadas: Uso de listas não ordenadas (<ul>) para os ingredientes e listas ordenadas (<ol>) para o passo a passo do modo de preparo.

Tags de Destaque: Aplicação de <strong> para enfatizar tempos de preparo, rendimento e dicas.

Inserção de Mídia: Uso correto da tag <img> com o atributo alt preenchido adequadamente para garantir a acessibilidade de leitores de tela.

Em CSS3:
Reset Básico: Zerar margens e paddings padrões dos navegadores (box-sizing: border-box).

Paleta de Cores: Escolha de tons quentes e gastronômicos (como marrom chocolate, cremes e tons suaves de fundo).

Tipografia: Configuração de fontes legíveis, tamanhos apropriados e espaçamento entre linhas (line-height).

Design de Cartão (Card Layout): Centralização e envolvimento do conteúdo em um container principal com bordas arredondadas e sombras suaves (box-shadow).
